# CAFFIN - Fashion Marketplace

## Deskripsi
CAFFIN merupakan website marketplace fashion berbasis Laravel yang dikembangkan sebagai proyek UAS mata kuliah Pemrograman Web. Website ini menyediakan sistem jual beli produk fashion secara online dengan tiga jenis pengguna, yaitu Admin, Seller, dan Buyer.

---

## Fitur Sistem

### 👤 Buyer
- Login & Register
- Melihat Produk
- Detail Produk
- Keranjang Belanja
- Checkout
- Wishlist
- Review Produk
- Chat Seller

### 🛍️ Seller
- Dashboard Penjualan
- CRUD Produk
- Kelola Pesanan
- Statistik Penjualan

### 👨‍💻 Admin
- Dashboard Admin
- Kelola User
- Kelola Kategori
- Kelola Pesanan

## Cara Menjalankan Project

1. Clone repository
2. Jalankan:

```bash
composer install
```

3. Jalankan:

```bash
npm install
```

4. Salin file:

```
.env.example
```

menjadi

```
.env
```

5. Jalankan:

```bash
php artisan key:generate
```

6. Import database

```
caffin_db.sql
```

7. Jalankan:

```bash
php artisan storage:link
```

8. Jalankan:

```bash
php artisan serve
```

9. Buka browser

```
http://localhost:8000
```
